# Running Locally

## Requirements
- Node.js + npm

## Minimal Web UI (Phase 2)
```bash
npm install
npm run dev
```

## Optional
```bash
npm run build
npm run preview
```

## Manual Testing Checklist (Phase 2)
- Play multiple full games
- Confirm blocked moves do not advance turns
- Confirm game-over disables input
- Confirm engine behavior matches Phase 1 tests
